* Matsuo Lab Deep Learning Course Exercises

Personal exercises and notes from the [[https://weblab.t.u-tokyo.ac.jp/deep-learning%e5%9f%ba%e7%a4%8e%e8%ac%9b%e5%ba%a7%e6%bc%94%e7%bf%92%e3%82%b3%e3%83%b3%e3%83%86%e3%83%b3%e3%83%84-%e5%85%ac%e9%96%8b%e3%83%9a%e3%83%bc%e3%82%b8/][Matsuo Lab Deep Learning Fundamentals Course]] at the University of Tokyo.

** Contents

| Chapter | Topic                                       | Format   |
|---------+---------------------------------------------+----------|
|       1 | Course introduction                         | org-mode |
|       2 | NumPy, Matplotlib, scikit-learn basics      | org-mode |
|       3 | Deep learning exercises                     | Jupyter  |
|       4 | Advanced topics                             | Jupyter  |

** Topics Covered

- NumPy array operations and broadcasting
- Matrix and tensor computations (matmul, tensordot, einsum)
- Random number generation
- Data visualization with Matplotlib
- Machine learning workflows with scikit-learn
- MNIST dataset handling and image classification

** Getting Started

Clone the repository and open the chapter files in Emacs (for org-mode) or Jupyter (for notebooks).

#+begin_src shell
git clone https://github.com/conao3/book-matsuo-lab-lecture.git
cd book-matsuo-lab-lecture
#+end_src

** Requirements

- Python 3.x
- NumPy
- Matplotlib
- scikit-learn
- Jupyter (for notebook chapters)
